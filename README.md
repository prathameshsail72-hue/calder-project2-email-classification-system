# 📧 Email Classification System
### Calder AI/ML Internship — Project 2
**Prathamesh Sail**

## Overview
End-to-end ML system classifying emails into Inquiry, Complaint, or Feedback using Logistic Regression + TF-IDF on real Kaggle customer support data.

## Project Structure
```
calder-email-classifier/
├── data/
│   ├── customer_support_tickets.csv   ← Raw Kaggle dataset (add manually)
│   └── emails_clean.csv               ← Generated by prepare_dataset.py
├── model/
│   ├── classifier.pkl                 ← Generated by classifier.py
│   └── vectorizer.pkl                 ← Generated by classifier.py
├── src/
│   └── classifier.py
├── app.py
├── prepare_dataset.py
├── requirements.txt
└── README.md
```

## Run Order
```bash
pip install -r requirements.txt
python prepare_dataset.py
python src/classifier.py
streamlit run app.py
```

## Label Mapping
| Original | Mapped |
|---|---|
| Product inquiry + Billing inquiry | Inquiry |
| Technical issue + Refund + Cancellation | Complaint |
| Synthetic emails | Feedback |
